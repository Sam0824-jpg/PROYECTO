## 📝 Descripción del Proyecto
Backend desarrollado en Node.js, Express y MySQL para la aplicación "FyNe". Este sistema gestiona un marketplace de oficios locales, permitiendo conectar a clientes con prestadores de servicios verificados (plomeros, electricistas, etc.).

## 📦 Módulos del Sistema (Entidades Reales)
El proyecto cuenta con un sistema de Autenticación JWT y 5 módulos CRUD completos:
1. **Usuarios:** Gestión de cuentas, clientes y administradores.
2. **Prestadores:** Perfiles de los trabajadores de oficios.
3. **Categorías:** Clasificación de los oficios (Plomería, Carpintería, etc).
4. **Servicios:** Catálogo de trabajos y precios que ofrece cada prestador.
5. **Reseñas:** Calificaciones y comentarios dejados por los clientes.
